# 🎙️ WhisperFlow — голосовой ввод для Windows

Аналог Whisper Flow: зажми CapsLock, надиктуй, отпусти — текст появится там, где стоит курсор.

## Установка

### 1. Установи зависимости

```bash
pip install -r requirements.txt
```

### 2. Запусти

```bash
python main.py
```

При первом запуске автоматически скачается модель Whisper (~500 МБ). Это займёт пару минут.

---

## Использование

1. Кликни мышкой в любое текстовое поле (браузер, VS Code, мессенджер и т.д.)
2. **Зажми CapsLock** — иконка в трее станет красной, запись началась
3. Говори на русском
4. **Отпусти CapsLock** — через 1–3 секунды текст появится в поле

## Настройки (`config.py`)

| Параметр | По умолчанию | Описание |
|---|---|---|
| `HOTKEY` | `'caps lock'` | Хоткей для записи |
| `WHISPER_MODEL` | `'small'` | Размер модели: `tiny` / `base` / `small` / `medium` |
| `WHISPER_LANGUAGE` | `'ru'` | Язык распознавания |

### Модели Whisper

| Модель | Размер | Скорость | Качество |
|---|---|---|---|
| `tiny` | ~75 МБ | очень быстро | базовое |
| `base` | ~150 МБ | быстро | хорошее |
| `small` | ~500 МБ | средне | **отлично** ✅ |
| `medium` | ~1.5 ГБ | медленно | превосходно |

### Сменить хоткей

В `config.py` измени `HOTKEY`. Примеры:
- `'right ctrl'` — правый Ctrl
- `'f9'` — клавиша F9
- `'`'` — тильда

---

## Автозапуск (опционально)

Чтобы WhisperFlow стартовал вместе с Windows:

1. Создай ярлык для `main.py`
2. Нажми `Win + R`, введи `shell:startup`
3. Перенеси ярлык в открывшуюся папку
